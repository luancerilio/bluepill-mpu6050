# MPU6050

Getting MPU6050 gyro and accelerometer data via I2C protocol. Data visualization via Serial.

## I2C
```
- SDA: PB9
- SCL: PB8
```
## Serial
```
- RX: PB7
- TX: PB6
- Baud Rate: 115200
```
